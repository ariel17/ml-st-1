DS9¶

DS9 states for Deep Space 9, a Federation station orbiting Bajor. In this project it works as the API implementation for weather web service requested by Vulcano.

class ds9.Weather[source]¶

It is the API resource providing weather information.

get()[source]¶

GET verb implementation. It receives a non-mandatory parameter called day, as follows:

  • /weather: Retrieves all existent predicted weather in database, with some metadata as total days analyzed and peak rain day.
  • /weather?day=N: Returns the forecast predicted for the indicated day, if it exists on database.
